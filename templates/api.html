API Documentation

GET /api/v1.0/motor_collision

Returns a list of all motor collisions.

Endpoint: /api/v1.0/motor_collision

GET /api/v1.0/motor_collision/total_count

Returns the total count of motor collisions.

Endpoint: /api/v1.0/motor_collision/total_count

GET /api/v1.0/motor_collision/<zip_code>

Returns all motor collision records filtered by ZIP code.

Endpoint: /api/v1.0/motor_collision/<zip_code>

GET /api/v1.0/weather_collision/<zip_code>

Returns weather collision data filtered by ZIP code.

Endpoint: /api/v1.0/weather_collision/<zip_code>

GET /api/v1.0/motor_statistics

Returns statistics for motor collisions.

Endpoint: /api/v1.0/motor_statistics

GET /api/v1.0/motor_collision/<zip_code>/by_weekday

Returns motor collision records by weekday filtered by ZIP code.

Endpoint: /api/v1.0/motor_collision/<zip_code>/by_weekday

GET /api/v1.0/motor_collision/<zip_code>/by_hour

Returns motor collision records by hour filtered by ZIP code.

Endpoint: /api/v1.0/motor_collision/<zip_code>/by_hour

GET /api/v1.0/motor_collision/<zip_code>/by_month

Returns motor collision records by month filtered by ZIP code.

Endpoint: /api/v1.0/motor_collision/<zip_code>/by_month

GET /api/v1.0/motor_collision/by_month

Returns motor collision records aggregated by month.

Endpoint: /api/v1.0/motor_collision/by_month

GET /api/v1.0/motor_collision/monthly_average

Returns the monthly average of motor collisions.

Endpoint: /api/v1.0/motor_collision/monthly_average

GET /api/v1.0/motor_collision/total_count

Returns the total count of motor collisions.

Endpoint: /api/v1.0/motor_collision/total_count

GET /api/v1.0/motor_collision/total_deaths

Returns the total number of deaths from motor collisions.

Endpoint: /api/v1.0/motor_collision/total_deaths

GET /api/v1.0/motor_collision/total_injuries

Returns the total number of injuries from motor collisions.

Endpoint: /api/v1.0/motor_collision/total_injuries

GET /api/v1.0/collision_by_zip

Returns collision records grouped by ZIP code.

Endpoint: /api/v1.0/collision_by_zip
